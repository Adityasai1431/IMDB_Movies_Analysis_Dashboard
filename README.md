# 🎬 IMDB Movies Dashboard - Power BI Project

## 📌 Project Overview
This project is a complete **Movie Sales & Analytics Dashboard** built using **Power BI**.  
It provides insights into movie trends, genre performance, revenue analysis, and advanced correlations using interactive visualizations.

---

## 📊 Key Features

### 🔹 1. Movie Overview & Time Trends
- Total Movies Released: **1000**
- Average IMDb Rating: **6.72**
- Average Runtime: **113 minutes**
- Total Revenue: **72.34K (Millions)**

📈 Insights:
- Movie releases increased significantly after 2014
- Revenue shows a consistent upward trend
- Ratings remained relatively stable over the years
- Runtime fluctuates between 100–120 minutes

---

### 🔹 2. Genre & Rating Analysis
- Total Genres: **20**
- Most Common Genre: **Drama**
- Highest Rated Genre: **War**
- Avg Metascore: **58.99**

📈 Insights:
- **Drama** dominates in number of movies
- **Adventure & Action** generate highest revenue
- Higher metascore does not always mean higher revenue

---

### 🔹 3. Top Performers & Box Office Insights
- Highest Revenue: **936.63M**
- Avg Revenue: **82.96M**
- Total Actors: **996**
- Total Votes: **170M**

🏆 Top Directors:
- Christopher Nolan
- J.J. Abrams
- David Yates

📈 Insights:
- Top movies generate a large share of total revenue
- High vote count strongly correlates with popularity

---

### 🔹 4. Correlation & Advanced Metrics
- Avg Revenue: **82.96M**
- Avg Metascore: **58.99**
- Avg Runtime: **113.17 mins**

📈 Insights:
- Positive correlation between **votes & revenue**
- Moderate relationship between **rating & revenue**
- Runtime has minimal impact on revenue
